# Data

These JSONL files contain the release-sized seed pools used by the anonymous
submission pipelines:

- `integration_seeds.jsonl`: 165 Tongji textbook integration seeds for
  hard-verifier setter training.
- `general_math_seeds.jsonl`: 1,711 easy MATH seeds used by the
  soft-verifier setter RL run, selected by solver pass rate from the seed pool.
